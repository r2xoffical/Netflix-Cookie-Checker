🚀 Özellikler
Gelişmiş Tema Motoru: 20'den fazla neon ve glow efektli terminal teması (Netflix Classic, Cyberpunk, Matrix vb.).

Yüksek Performans: Özelleştirilebilir Multi-threading desteği (1-100 thread).

Akıllı Loglama: Sonuçları tarih ve saat damgalı olarak otomatik kaydeder.

Cookie Desteği: cookies/ klasöründeki verileri otomatik algılar ve işler.

Hata Yönetimi: Geçersiz dosyalar ve ağ hataları için gelişmiş kontrol mekanizması.

🎨 Arayüz Önizlemesi
Yazılımın terminal üzerindeki görüntüsü, dinamik renk geçişleri ve ASCII banner tasarımlarıyla optimize edilmiştir.

(Buraya terminalden aldığın bir ekran görüntüsünü ekleyebilirsin)

🛠️ Kurulum
Depoyu klonlayın:

Bash
git clone [https://github.com/r2xzzs/netflix-checker.git](https://github.com/r2xzzs/netflix-checker.git)
Gerekli kütüphaneleri yükleyin:

Bash
pip install requests pyyaml
Uygulamayı başlatın:

Bash
python main.py
⚙️ Yapılandırma
main.py içerisindeki THEMES sözlüğü üzerinden renkleri ve efektleri manuel olarak düzenleyebilirsiniz. Thread ayarları uygulama başlangıcında dinamik olarak seçilebilir.
